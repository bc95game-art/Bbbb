# 🔗 استخراج لینک (Link Extractor)

برنامه اندروید برای شناسایی و استخراج لینک‌های پنهان از سایر برنامه‌ها.

## چطور کار می‌کند؟

1. **Accessibility Service** — لینک‌های HTTP/HTTPS را از محتوای صفحه‌نمایش اپ‌های هدف استخراج می‌کند
2. **Floating Window** — یک دکمه شناور روی اپ‌ها نمایش می‌دهد که می‌توانید آن را جابجا کنید
3. **یک کلیک** — با کلیک روی «استخراج»، لینک کشف‌شده مستقیماً در کلیپ‌بورد ذخیره می‌شود

## راه‌اندازی پروژه در Android Studio

### پیش‌نیاز
- Android Studio Hedgehog یا جدیدتر
- JDK 17 یا بالاتر
- Android SDK (minSdk 26 / Android 8.0)

## مراحل استفاده

### ۱. اجرای برنامه
برنامه را نصب و باز کنید.

### ۲. فعال‌سازی مجوزها
- **مجوز نمایش روی برنامه‌ها**: روی دکمه کلیک کنید و برنامه را فعال کنید
- **سرویس دسترس‌پذیری**: روی دکمه کلیک کنید، «استخراج لینک» را پیدا کرده و فعال کنید

### ۳. انتخاب برنامه‌های هدف
روی «انتخاب برنامه‌های هدف» کلیک کرده و برنامه‌هایی مانند **شاد**، **ذربین** و غیره را انتخاب کنید.

### ۴. شروع سرویس
روی «شروع سرویس شناور» کلیک کنید. یک اعلان دائمی نشان می‌دهد سرویس فعال است.

### ۵. استخراج لینک
- وارد برنامه هدف شوید و صفحه مورد نظر را باز کنید
- دکمه شناور «🔗 استخراج» را ببینید
- روی آن کلیک کنید — لینک استخراج شده و در کلیپ‌بورد کپی می‌شود!

## ساختار پروژه

```
android-app/
├── app/src/main/java/com/linkextractor/app/
│   ├── MainActivity.kt           — صفحه اصلی + کنترل مجوزها
│   ├── AppSelectorActivity.kt    — انتخاب برنامه‌های هدف
│   ├── FloatingWindowService.kt  — سرویس دکمه شناور
│   ├── LinkAccessibilityService.kt — سرویس دسترس‌پذیری (استخراج لینک)
│   ├── AppAdapter.kt             — لیست برنامه‌ها
│   ├── LinkAdapter.kt            — لیست لینک‌های استخراج‌شده
│   ├── PrefsManager.kt           — ذخیره تنظیمات و تاریخچه
│   └── AppInfo.kt                — مدل داده برنامه
├── app/src/main/res/
│   ├── layout/                   — فایل‌های XML رابط کاربری
│   ├── values/                   — رشته‌ها، رنگ‌ها، تم
│   └── xml/                      — تنظیمات سرویس دسترس‌پذیری
└── app/src/main/AndroidManifest.xml
```

## مجوزهای لازم

| مجوز | دلیل |
|------|------|
| `SYSTEM_ALERT_WINDOW` | نمایش دکمه شناور روی سایر برنامه‌ها |
| `FOREGROUND_SERVICE` | نگه‌داشتن سرویس فعال |
| `BIND_ACCESSIBILITY_SERVICE` | خواندن محتوای صفحه برنامه‌ها |
| `QUERY_ALL_PACKAGES` | نمایش لیست برنامه‌های نصب‌شده |

## نکات مهم

- این برنامه فقط برای **استفاده شخصی و آموزشی** طراحی شده
- در برخی دستگاه‌ها ممکن است نیاز به مجوز اضافی باشد (مثل MIUI/EMUI)
